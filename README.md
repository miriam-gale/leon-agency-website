# Leon Agency – Minimal Web Template

Leon Agency is a clean, minimal, and responsive web template built using **HTML** and **CSS**.  
It is inspired by modern agency landing pages and focuses on simplicity, layout, and good visual hierarchy.

This project was built as a hands-on practice to strengthen core front-end skills such as layout design, positioning, responsiveness, and clean styling.
